def count_elements(seq) -> dict:
     """Tally elements from `seq`."""
     hist = {}
     for i in seq:
         hist[i] = hist.get(i, 0) + 1
     return hist

This is a function that counts the frequency of the data in the given column and stores it in form of dictionary
